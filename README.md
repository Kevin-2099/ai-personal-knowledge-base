# 🧠 AI Personal Knowledge Base

Una base de conocimiento personal que conecta automáticamente tus notas, ideas y proyectos usando IA semántica.

---

## 🚀 ¿Qué es esto?

Es un sistema tipo “segundo cerebro digital” donde puedes:

- Guardar notas y documentos
- Encontrar relaciones entre ideas
- Explorar un mapa visual de conocimiento
- Detectar temas y patrones automáticamente
- Hacer preguntas sobre tu propio conocimiento

---

## 🧠 Cómo funciona

1. **Añades notas**
   - Texto libre (ideas, proyectos, apuntes)

2. **IA local analiza el significado**
   - Usa `SentenceTransformers`
   - Convierte texto en embeddings

3. **Conecta ideas automáticamente**
   - Calcula similitud semántica
   - Detecta relaciones entre conceptos

4. **Construye un mapa mental**
   - Grafo interactivo de conocimiento
   - Visualización de conexiones

5. **Extrae insights**
   - Temas principales
   - Conocimiento repetido
   - Relación entre áreas
